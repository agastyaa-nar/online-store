## Instructions:

1. Railway Backend Setup:
   - Go to Railway Dashboard
   - Click "online-store" service
   - Go to "Variables" tab
   - Add each variable above (copy name and value)

2. Database Connection:
   - Click "Connect Database" in online-store service
   - Select PostgreSQL service
   - Railway will auto-generate DATABASE_URL

3. Database Schema Setup:
   - Go to PostgreSQL service
   - Click "Console"
   - Copy entire content from database/schema.sql
   - Paste and execute in console

4. Frontend Configuration (Vercel):
   - Go to Vercel Dashboard
